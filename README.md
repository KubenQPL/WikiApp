# WikiApp
WikiApp jest przykładową aplikacją pokazującą użycie bibliotek takich jak: Retrofit, RxKotlin, Dagger; jak również użycie wzorca projektowego Model-View-Presenter.Aplikacja nie jest jeszcze skończona.
